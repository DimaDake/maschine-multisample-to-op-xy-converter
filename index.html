Multisample Preset Converter

Convert your instrument libraries into presets for your synthesizer.

Instructions

  1. Click the Select Instrument Library button.
  2. Your browser will ask for permission to access a folder. Choose your main instruments directory (e.g., `.../Samples/Instruments`).
  3. The application will scan for instrument sub-folders containing .wav files.
  4. For each instrument found, it will generate a .preset.zip file with resampled audio.
  5. Download links for each generated preset will appear in the "Results" section.
